About:

I’ve been a software tester for about 12 years, during which time I became somewhat skilled in using perl. I’ve always dreamed of writing my own bugs instead of hunting down those of others. I’m no longer testing software, and with a few web sites under my belt (written in PHP), I’m thinking there must be a better way. I can’t wait to write some really complex ruby code. The fact that testing is practically built in is really interesting to me. Unfortunately I’m writing code only part-time, but I hope that will soon change. Until then I keep plugging along.
